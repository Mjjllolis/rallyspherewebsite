# Photo sources & usage

Web-optimized from `~/Downloads/High Res/`. To swap an image, replace the file
here (keep the same filename) or drop a new source and re-run the optimize step.
Re-optimize command (from project root):

```
NODE_PATH=node_modules node scripts/optimize-photos.js   # if saved; otherwise see chat history
```
